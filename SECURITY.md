# Security

CodexLens is designed to be local-only and read-only.

Please report security issues privately by opening a GitHub security advisory or contacting the repository owner. Do not disclose sensitive local log contents in public issues.

When reporting a problem, include:

- your macOS version
- Node.js version
- CodexLens version or commit
- a minimal description of the affected behavior

Do not include API keys, auth files, prompt text, private repository names, or raw session logs unless you have reviewed and redacted them.
